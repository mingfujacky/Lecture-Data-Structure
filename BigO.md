# Sigma Calculation

$\sum_{i=1}^n 1 = n$ <br>
$\sum_{i=1}^n c = n * c$ <br>
$\sum_{k=j}^n 1 = n - j + 1$ <br>
$\sum_{i=j}^n i = \frac{n(n+1)}{2} - \frac{j(j-1)}{2}$

$\sum_{i=1}^n \big(a + b\big) = \sum_{i=1}^n a + \sum_{i=1}^n b$
$\sum_{i=1}^n \big(c * a\big) = \sum_{i=1}^n a + \sum_{i=1}^n b$
